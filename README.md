# 🎬 Video Summarization API

A full-stack application that automatically summarizes videos using deep learning. Upload a video through the frontend interface and receive a concise summary generated by a Python-powered backend API.

---

## 🗂️ Project Structure

video_summarization_api/
├── backend/        # FastAPI server with video processing & summarization logic
├── frontend/       # JavaScript-based UI for uploading videos and viewing results
├── layers/         # Shared/utility layers (model weights, helper modules)
├── requirements.txt
└── README.md


---

## 🚀 Features

- Upload videos via a clean web interface
- Backend processes video frames using computer vision (OpenCV, PyTorch)
- Returns a summarized version of the video or key frames
- RESTful API built with FastAPI
- Fast inference using PyTorch and torchvision

---

## 🛠️ Tech Stack

**Backend**
- [FastAPI](https://fastapi.tiangolo.com/) — REST API framework
- [PyTorch](https://pytorch.org/) — Deep learning model inference
- [OpenCV](https://opencv.org/) — Video frame extraction
- [scikit-learn](https://scikit-learn.org/) — Feature processing
- [Pillow](https://python-pillow.org/) — Image handling
- [h5py](https://www.h5py.org/) — Model/data storage

**Frontend**
- JavaScript, HTML, CSS — Browser-based upload interface

## ⚙️ Installation

### Prerequisites
- Python 3.9+
- pip

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/imran2600/video_summarization_api.git
cd video_summarization_api

# 2. Install dependencies
pip install -r requirements.txt

# 3. Start the backend server
cd backend
uvicorn main:app --reload
```

The API will be available at `http://127.0.0.1:8000`

---

## 📡 API Usage

Once the server is running, visit `http://127.0.0.1:8000/docs` for the interactive Swagger UI to explore and test all available endpoints.

**Example endpoint:**
POST /summarize
Content-Type: multipart/form-data
Body: video file (mp4, avi, etc.)
---

## 🖥️ Frontend

Open the `frontend/` folder in your browser or serve it with any static file server:

```bash
cd frontend
npx serve .
```
